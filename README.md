# 🏥 Sistema de Gestão de Clínica Veterinária

Este é um sistema robusto desenvolvido em Java com o objetivo de automatizar e otimizar os processos diários de uma clínica veterinária. O projeto foi estruturado aplicando os conceitos fundamentais da Programação Orientada a Objetos (POO) para resolver problemas reais de agendamento, gestão de clientes e controlo de prontuários médicos de animais.

---

## 🚀 Funcionalidades Principais

* **🐾 Gestão de Clientes e Pets:** Registo completo de tutores e a associação direta aos seus respetivos animais (nome, espécie, raça, idade e histórico).
* **📅 Controlo de Consultas:** Agendamento, atualização e histórico completo de atendimentos veterinários.
* **💼 Prontuário Médico Digital:** Registo detalhado de diagnósticos, tratamentos aplicados, vacinas e observações clínicas para cada animal.
* **🔍 Filtros e Pesquisas:** Localização rápida de dados para otimizar o tempo de atendimento da clínica.

---

## 🛠️ Tecnologias e Conceitos Aplicados

* **Linguagem:** Java (JDK 21)
* **Paradigma:** Programação Orientada a Objetos (Abstração, Encapsulamento, Herança e Polimorfismo)
* **Estrutura de Dados:** Coleções nativas do Java (`List`, `ArrayList`) para manipulação de dados em memória.
* **IDE:** IntelliJ IDEA Community Edition
